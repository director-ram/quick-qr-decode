# HAG's QR Scanner & Generator

A modern, interactive QR code scanner and generator web application built with React, TypeScript, and Tailwind CSS.

## Features

üîç **QR Code Scanner**
- Real-time camera scanning using html5-qrcode
- Support for various QR code formats (URLs, WiFi, contacts, etc.)
- Automatic format detection and parsing
- Copy scanned data to clipboard
- Smart action buttons (open URLs, dial numbers, etc.)

üé® **QR Code Generator**
- Generate QR codes from multiple data types:
  - Plain text
  - Website URLs
  - WiFi network credentials
  - Contact cards (vCard)
  - Email addresses
  - SMS messages
- High-quality canvas rendering
- Download generated QR codes as PNG
- Copy data to clipboard

üì± **Modern UI/UX**
- Beautiful glass morphism design
- Animated backgrounds and floating particles
- Responsive layout for all devices
- Smooth animations and transitions
- Dark theme with purple/pink gradients

üíæ **Smart History**
- Persistent history using Firebase
- User IP-based storage
- Track both generated and scanned QR codes
- Export and manage history
